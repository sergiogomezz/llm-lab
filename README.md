# llm-lab

Personal lab for understanding LLMs from the ground up — architecture, 
pretraining, fine-tuning and inference.

## Structure

- `pretraining/` — building a GPT-like model from scratch
- `finetuning/` — fine-tuning experiments on pretrained models
- `experiments/` — miscellaneous explorations
